# looping-pemdas

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Aditya27T/looping-pemdas.git
~~~

Go to the project directory  

~~~bash  
  cd looping-pemdas
~~~

Start the cpp file  

~~~bash  
  g++ <filename> -o <output name>
~~~

## Usage/Examples

~~~bash
  g++ hello.cpp -o hai
~~~  

~~~bash
  ./hai
~~~
